Справка по параметрам командной строки
Параметры по порядку:
Режим работы:
1   Режим обучения;
        2   Режим распознавания.
Входной путь:
        В первом режиме путь к тестовой выборке.
        Во втором режиме путь к файлу с обученной сетью.
Третий параметр:
В первом режиме путь для сохранения обученной сети;
Во втором режиме путь к папке с тестовой выборкой объектов.
следующие параметры существуют только в первом режиме:
Скорость обучения;
Тип функции активации:
        1    Логистическая функция задаётся одним следующим параметром
        2   Гиперболический тангенс задаётся двумя следующими параметрами
        3   задаётся двумя следующими параметрами
    Количество классов в выходном слое
    Следующие параметры задают количество нейронов на скрытых слоях, количество следующих параметров задаёт количество скрытых слоёв.
